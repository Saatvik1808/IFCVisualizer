What is IFC?

ISO

Industry Foundation Classes

IFC is an open, international standard (ISO 16739) for sharing BIM data across software platforms.

SH

Spatial Hierarchy

Buildings are organized as: Project → Site → Building → Storey → Space → Elements

REL

Entity Relationships

Entities reference each other via IDs (#123). Relationships like IfcRelAggregates define containment.

STP

STEP Format

IFC uses ISO 10303 STEP encoding. Each line defines an entity: #id= IFCTYPE(args);
